# 🎭 Sarcasm Detection Service

Сервис по классификации английских заголовков новостей на предмет сарказма. 
Проект выполнен в рамках домашнего задания по машинному обучению.

## 📊 Исследование и выбор модели
В ходе экспериментов в Google Colab были протестированы 4 модели:
1. **Logistic Regression** (Baseline)
2. **Naive Bayes**
3. **Random Forest**
4. **XGBoost**

### Результаты тестирования:
| Model | F1-score | Inference Speed (s) |
| :--- | :---: | :---: |
| **Logistic Regression** | **0.8204** | **0.0004** |
| Naive Bayes | 0.8119 | 0.0006 |
| Random Forest | 0.7920 | 0.5519 |
| XGBoost | 0.7911 | 0.0197 |

**Вывод:** Для финального сервиса выбрана **Logistic Regression**, так как она показала лучший баланс точности (F1-score) и экстремально высокую скорость работы (Inference), что критично для веб-сервисов.

## 🛠 Архитектура проекта
Проект организован по принципам ООП и модульности:
* `model_utils.py`: Содержит класс `SarcasmClassifier` для загрузки модели и обработки текста.
* `main.py`: API-интерфейс на базе **FastAPI**.
* `Dockerfile`: Инструкция для контейнеризации приложения.
